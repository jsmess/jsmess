Options Menu

       Game List Font :  Selecting this option from the options menu will change the font used for the GUI display.

       Game List Clone Color :  Selecting this option from the options menu will change the font color used for the GUI display of clones in the 'Grouped' game list view.

       Directories : Selecting this option from the options menu will allow for choosing a different directory for a number of MameUI support files including: ROMs, samples, config, high scores, snapshots, input files, state, artwork, memory card, flyers, cabinets, marquees, and nvram.  Normally these directories are created under the MameUI executable’s directory.

       Default Options : Selecting this option from the options menu will allow for the global customization of all games in the game list.  These settings are overridden by altering the individual game settings using the File • Properties menu item.

       Reset to Default : Selecting this option from the options menu will allow for resetting various parts of MameUI to their defaults.  MameUI will prompt before resetting, giving the option to cancel.

       Interface Options : Selecting this option from the options menu will allow:
Toggling on or off the search for new games on start.
Toggling on or off the ability to use a game pad to navigate the GUI (see mame32ui.ini for examples).
Toggling on or off the ability to use a keyboard encoder solution to navigate the GUI and launch games [helpful in Mame cabs] (see mame32ui.ini for examples).
Selecting a random image from the \bkground directory on startup.
Toggling on or off MameUI broadcasting the selected game programmatically to other listening applications like message clients, or an LED message bar,
The cursor to be hidden [helpful in Mame cabs].
Selecting inheritable filters, this allows parent folders to set the filters for their children sub-folders.
No offset for clones missing parent in view toggle will cause clones not be indented if the parent game is not present due to current folder view, missing ROMs, or filtering.
Allow history.dat file to be displayed under any, all, or none of the art views.
Show game caption toggles the titlebar and controls in windowed mode, if off, only a single frame will surround the playing game.  If on, the full titlebar will appear complete with icon and name of the game being played.
Globally clear the play time and number of times played counters.
Stretch screenshot to larger size, this will enlarge the items in the art pane scaling them upwards depending on the size of the GUI.
Screenshot border size and color creates a user defined border around the art work in the art pane on the right, set to zero to remove borders completely.

 

 

       Background Image : Selecting this option from the options menu allows the choosing of a background image from the \bkground directory.  This image will be displayed beneath the three window panes in the GUI.
