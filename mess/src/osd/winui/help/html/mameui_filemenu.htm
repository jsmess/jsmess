File Menu

       Play Game : Selecting this option from the file menu will launch the highlighted game.

       Play and Record Input : Selecting this option from the file menu will allow for the creation of an *.inp file which will allow MameUI to record keystrokes in the game being played for playback later.  Similar to a VCR.

       Playback Input : Selecting this option from the file menu will allow the loading and playing of a previously recorded *.inp file.

       Play and Record Wave Output : Selecting this option from the file menu will allow the recording of a sound file while playing a game.  The *.wav file is placed in the \inp directory by default.

       Play and Record MNG Output : Selecting this option from the file menu will allow the recording of a MNG video file while playing a game.  The *.mng file is placed in the \inp directory by default.

       Load Savestate : Selecting this option from the file menu will allow the loading and playing of a previously saved machine state, the state being loaded must match the highlighted game.

       Select Random Game : Selecting this option from the file menu will cause MameUI to randomly highlight a game in the game list shortcut key is Ctrl-R.

       Properties : Selecting this option from the file menu will display the highlighted game’s individual settings, these settings over-ride those made to all games in the Options • Default Options menu item. 

       Audit All Games : Selecting this option from the file menu will initiate an exhaustive check on the available games, verifying them to MameUI’s expected results.

       Exit : Selecting this option from the file menu will exit the program.
